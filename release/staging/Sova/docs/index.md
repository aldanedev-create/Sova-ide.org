# Sova Documentation

Sova 0.1v is a readable, dynamically typed general-purpose language with
optional annotations. Source follows one pipeline: lexer, parser, AST, semantic
analysis, then the tree-walking runtime.

Start with [Getting Started](getting-started.md), then study variables,
collections, control flow, functions, classes, modules, error handling, and the
standard library. Use [Online IDE](online-ide.md) for restricted browser code or
[Installation](installation.md) for the complete local CLI.

Sova is implemented using NovaDev but is not a NovaDev app mode. It owns its
grammar, runtime behavior, diagnostics, CLI, and release lifecycle.

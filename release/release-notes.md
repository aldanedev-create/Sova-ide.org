# Sova 0.1.0

Initial public binary release of the Sova interpreter, CLI, interactive shell,
standard-library namespace, official website, browser IDE, restricted Vercel
API, Windows installer, and portable Windows distribution.

This build also includes rich `.sova` highlighting, snippets, indentation rules,
and Sova file icons for VS Code, VSCodium, Cursor, and Windsurf.

The Windows artifacts are unsigned, so Windows SmartScreen may display an
unrecognized-publisher warning. Verify downloads with the published SHA-256
checksums before installation.

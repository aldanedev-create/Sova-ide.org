# Roadmap

## Implemented In 0.1v

- source-located lexer and one precedence parser
- complete AST and structural semantic pass
- dynamic runtime with optional outer-type checks
- functions, closures, classes, single inheritance, and modules
- modern collections, safe access, and error handling
- local and restricted-online policies with execution limits
- CLI, shell, examples, tests, and standard-library aliases

## Stabilization

- improve unused-variable and unused-import source tracking
- broaden interpolation diagnostics
- add package publishing and version constraints
- strengthen generic item enforcement
- add deterministic formatter comments/string awareness

## Planned

- bytecode compiler and stack VM
- async/await, iterators, generators, enums, traits/interfaces, and pattern matching
- optional native extensions and carefully scoped FFI
- WebAssembly and optional ahead-of-time compilation research

## Not Promised For 0.1v

Multiple inheritance, raw pointers, manual memory, templates, native C++ performance, and Rust-style borrow checking are not current Sova features.
